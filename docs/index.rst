********************************
RHEAS Viewer 1.0
********************************

The RHEAS Viewer is a web application for viewing RHEAS(Regional Hydrologic Extremes Assessment System) output data. This app was developed using the Tethys Platform. 

To learn more about RHEAS, see: http://rheas.readthedocs.io/en/latest/. 

To learn more about the Tethys Platform see: http://docs.tethysplatform.org/en/stable/index.html.
